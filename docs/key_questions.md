# Ответы на ключевые вопросы

## Проблема

Организация программного решения повторной идентификации человека на видеопотоке с камер видеонаблюдения в условиях перехода от дневного к ночному режиму съемки.

## Актуальность

Активное распространение систем видеонаблюдения и круглосуточного мониторинга создает потребность в автоматическом анализе видеоданных, в частности, для непреривный идентификации объектов вне зависимости от времени суток [1-3]. Большинство современных камер видеонаблюдения автоматически переключаются в режим инфракрасной съемки (IR-режим) в ночное время суток, что делает неработоспособными стандартные алгоритмы, ориентированные на анализ цвета - точность базовых моделей при смене спектра падает на 30–40% [4]. Существующие методы используют различные подходы решения этой проблемы (кросс-модальная идентификация), такие как генеративные сети GAN, механизмы внимания, аугментация данных и другие [5]. Но большинство описанных в научных публикациях методов представленны в виде схем и не реализованы в виде полноценных программных модулей, следовательно трудно применимы на практике. Поэтому существует потребность в реализации программного модуля, повышающего устойчивость повторной идентификации к переключению RGB-IR-режима съемки.

## Объект исследования

Повторная идентификация человека в видеопотоке с распределенных камер видеонаблюдения.

## Предмет исследования

Устойчивость методов повторной идентификации человека в видеопотоке к условиям изменения модальности.

## Цель дипломного исследования

Разработка программного модуля для повторной идентификации человека, устойчивого к изменению модальности.

## Цель на текущий семестр

Обзор существующих методов кросс-модальной повторной идентификации для определения подходов, наиболее подходящих для разработки программного модуля, устойчивого к изменениям цветовой информации видеозаписи.

## Задачи

1. Провести сравнительный анализ существующих подходов к улучшению точности кросс-модальной идентификации.
2. Сформулировать критерии для оценки эффективности методов кросс-модальной идентификации.
3. По сформулированным критериям провести сравнительный анализ перспективных методов, устойчивых к изменению цветового канала видеозаписи.
4. По результатам сравнения выделить наиболее устойчивые подходы и сформулировать рекомендации по их использованию в качестве основы для разрабатываемого программного модуля.
